# v2.0.0 Layout Flip + Git Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current `out/<runId>/<slug>/` multi-run layout with a single canonical `out/<slug>/` per protocol, backed by a real git repo inside `out/` for version history.

**Architecture:** Add a thin `framework/version-store.mjs` (~80 lines) wrapping `git` shelled via `child_process`. Hook `ensureRepo` + auto-commit into `framework/orchestrator.mjs` so every successful crawl produces one git commit. Refactor `framework/out-browser.mjs` to walk the flat layout and surface git-log history. Old `out/<runId>/...` directories are left untouched (cold cut migration).

**Tech Stack:** Node.js (no new deps), `git` shelled out via `child_process.spawn`, zero-dep test runner at `tests/run.mjs`.

**Out of scope (deferred to v2.1.0):** All seven user-facing stage CLIs (`i18n`, `get`, `set`, `history`, `diff`, `restore`, `refresh`). v2.0 only ships the foundation; users still drive everything via `crawl`.

**Spec:** `docs/superpowers/specs/2026-04-27-decouple-stages-and-versioning-design.md` (commit `a16425d`).

## Architecture diagrams

### Commit-hook data flow (when does each git op fire?)

```
  bash run.sh --batch pendle --batch morpho --parallel 2
                         │
                         ▼
            ┌─────────────────────────────┐
            │  run() entry                │  ensureRepo(out) — idempotent,
            │   ensureRepo(out)  ◄────    │  fires once per invocation,
            └──────────────┬──────────────┘  no-op on existing .git
                           │
              ┌────────────┴────────────┐
              │ parallel-runner (lim=2) │
              └────┬────────────────┬───┘
                   │                │
         ┌─────────▼──────┐  ┌──────▼─────────┐
         │ runOne(pendle) │  │ runOne(morpho) │  ◄── parallel writes
         │ guardClobber() │  │ guardClobber() │      to per-slug dir.
         │ R0→R1→R2→...   │  │ R0→R1→R2→...   │      NO git ops here.
         └──────┬─────────┘  └──────┬─────────┘
                │                   │
                └─────────┬─────────┘
                          │ all providers settled
                          ▼
            ┌─────────────────────────────────┐
            │  for slug of okSlugs:           │  ◄── SEQUENTIAL. Commits
            │    commit(out, {                │      MUST stay post-parallel.
            │      paths: [`${slug}/`],       │      Inlining into runOne()
            │      message: `crawl(${slug})...│      races against .git/index.
            │      runId,                     │
            │    })                           │
            └──────────────┬──────────────────┘
                           │
                           ▼
            ┌─────────────────────────────┐
            │  appendRunsLog({runId,...}) │
            │  buildOutBrowser(out)       │
            └─────────────────────────────┘
```

### Slug-dir state machine

```
        ┌────────────────────────┐
        │   absent / no record   │ ◄── first crawl
        └───────────┬────────────┘
                    │ runOne writes record.json + siblings
                    ▼
        ┌────────────────────────┐
        │       dirty            │ ◄── working tree has uncommitted writes
        │ (writes done, no commit yet)
        └───────────┬────────────┘
                    │ commit() succeeds (post-parallel loop in run())
                    ▼
        ┌────────────────────────┐
        │       clean            │ ◄── canonical, in HEAD
        └───────┬─────────┬──────┘
                │         │
        user edits        next crawl
                │         │
                ▼         ▼
        ┌──────────┐  ┌──────────────────┐
        │  dirty   │  │ guardClobber()   │
        │(manual)  │  │  if not clean:   │
        └────┬─────┘  │    THROW unless  │
             │        │    --force-overwrite
             │        └──────────────────┘
             │
   user runs crawl → REJECTED
   (refuse-clobber invariant — the safety guarantee)
```

### Failed-pipeline invariant

```
runOne(pendle)
  ├─ guardClobber()  ✓ clean
  ├─ R0  ✓
  ├─ R1  ✗ FAILS
  └─ ── pipeline aborts, _debug/ has artifacts ──
                          │
                          ▼
            workerFailures includes pendle
            okSlugs does NOT include pendle
                          │
                          ▼
   for slug of okSlugs: commit(out, { paths:[slug+'/'], message, runId })
            └── pendle is NOT in okSlugs → no commit lands
                          │
                          ▼
   out/pendle/ is left at HEAD's prior state
   (or absent if first-ever crawl).
   NO commit pollutes history with a half-failed record.
```

```js
// framework/version-store.mjs — git wrapper for out/.git history layer.
//
// Pure shell-out via child_process.spawn. No libgit2, no isomorphic-git —
// `git` on PATH is already required for the plugin (installed via Claude
// Code's marketplace).

import { spawn } from 'node:child_process';
import { writeFile, access } from 'node:fs/promises';
import { existsSync } from 'node:fs';
import { join } from 'node:path';

const GITIGNORE_BODY = `_debug/
.runs/
.runs.log
index.html
`;

function git(args, { cwd, stdin = null } = {}) {
  return new Promise((resolve, reject) => {
    const proc = spawn('git', args, { cwd, stdio: ['pipe', 'pipe', 'pipe'] });
    let stdout = '';
    let stderr = '';
    proc.stdout.on('data', (b) => { stdout += b.toString('utf8'); });
    proc.stderr.on('data', (b) => { stderr += b.toString('utf8'); });
    proc.on('error', reject);
    proc.on('close', (code) => {
      if (code === 0) resolve({ stdout, stderr });
      else reject(new Error(`git ${args.join(' ')} exited ${code}: ${stderr.trim()}`));
    });
    if (stdin != null) {
      proc.stdin.write(stdin);
    }
    proc.stdin.end();
  });
}

export async function ensureRepo(outDir) {
  if (existsSync(join(outDir, '.git'))) return;
  await git(['init', '--quiet', '-b', 'main'], { cwd: outDir });
  await git(['config', 'user.email', 'protocol-info@local'], { cwd: outDir });
  await git(['config', 'user.name', 'protocol-info'], { cwd: outDir });
  await writeFile(join(outDir, '.gitignore'), GITIGNORE_BODY);
}
```

```js
export async function commit(outDir, { paths, message, runId }) {
  for (const p of paths) {
    await git(['add', '--', p], { cwd: outDir });
  }
  // Detect empty staging area: `git diff --cached --quiet` exits non-zero
  // when there ARE staged changes, so we invert.
  const hasStaged = await new Promise((resolve) => {
    const proc = spawn('git', ['diff', '--cached', '--quiet'], { cwd: outDir, stdio: 'ignore' });
    proc.on('close', (code) => resolve(code !== 0));
  });
  if (!hasStaged) return null;

  const args = ['commit', '--quiet', '-m', message];
  if (runId) args.push('--trailer', `Run-Id: ${runId}`);
  await git(args, { cwd: outDir });
  const { stdout } = await git(['rev-parse', '--short', 'HEAD'], { cwd: outDir });
  return stdout.trim();
}
```

```js
const LOG_SEP = '\x1f'; // ASCII Unit Separator
const LOG_REC = '\x1e'; // ASCII Record Separator

export async function log(outDir, { slug, limit = 50 }) {
  const format = ['%H', '%ct', '%s', '%(trailers:key=Run-Id,valueonly)'].join(LOG_SEP) + LOG_REC;
  let stdout = '';
  try {
    ({ stdout } = await git(
      ['log', `-${limit}`, `--format=${format}`, '--', `${slug}/`],
      { cwd: outDir }
    ));
  } catch (err) {
    // Empty repo: git log exits 128. Treat as no history.
    if (/does not have any commits yet|unknown revision/i.test(err.message)) return [];
    throw err;
  }
  return stdout
    .split(LOG_REC)
    .map((s) => s.trim())
    .filter(Boolean)
    .map((rec) => {
      const [sha, ct, message, runId] = rec.split(LOG_SEP);
      return {
        sha: sha.slice(0, 12),
        ts: new Date(Number(ct) * 1000).toISOString(),
        message,
        runId: runId.trim() || null,
      };
    });
}
```

```js
export async function collectOutIndex(outDir) {
  const protocols = [];
  let entries;
  try { entries = await readdir(outDir, { withFileTypes: true }); }
  catch { return { protocols: [], runsLog: [] }; }

  for (const ent of entries) {
    if (!ent.isDirectory()) continue;
    if (ent.name.startsWith('.')) continue;          // .runs, .git, ...
    if (/^\d{8}T\d{6}Z$/.test(ent.name)) continue;   // legacy run-id dirs
    const recordPath = join(outDir, ent.name, 'record.json');
    try { await stat(recordPath); }
    catch { continue; }
    protocols.push({
      slug: ent.name,
      recordPath,
      dir: join(outDir, ent.name),
    });
  }
  protocols.sort((a, b) => a.slug.localeCompare(b.slug));

  const runsLog = await readRunsLog(outDir);
  return { protocols, runsLog };
}

async function readRunsLog(outDir) {
  try {
    const body = await readFile(join(outDir, '.runs.log'), 'utf8');
    return body.trim().split('\n').filter(Boolean).map((line) => {
      const [ts, runId, slugs, outcome] = line.split('\t');
      return { ts, runId, slugs: slugs.split(','), outcome };
    });
  } catch { return []; }
}
```
